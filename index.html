Indigo Music

Indigo Music is a bot that lets you play music with features such as Ownership, Filters, Loops etc. Read more

Essential Commands

  • /play [song name/url] This command is used to play any song you wish to.
  • /pause This command is used to pause the current song.
  • /resume This command is used to resume the current song if it is paused.
  • /stop This command is used to stop any kind of ongoing music.
  • /volume This command is used to increase or decrease the volume of the bot.
  • /skip This command is used to skip the current playing song.
  • /seek [timestamp] This command is used to seek to a particular timestamp of the song.
  • /loop This command is used to loop the current playing song in the bot.
  • /disconnect This command is used to make the bot disconnect safely from your music channel.

Utility Commands

  • /autoplay This command is used to enable autoplay in the bot.
  • /transfer This command is used to transfer control of the bot to someone else in the voice channel.
  • /filter Adds chosen filters to the currently playing music.
  • /replay This command replays the current playing song.
  • /shuffle This command is used to shuffle the queue of songs in your voice session.
  • /redeem This command is used to redeem a premium code to make the bot work.

Information Commands

  • /nowplaying This command displays the current playing song.
  • /queue This command is used to view the current queue of songs in the bot.
  • /invite This command is used to send the invite link of the bot.
  • /stats This command is used to view the statistics of the bot.
